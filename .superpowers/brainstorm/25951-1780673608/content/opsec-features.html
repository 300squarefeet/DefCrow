OPSEC Features — Pilih Semua yang Diinginkan

Centang fitur OPSEC yang ingin dimasukkan ke dalam loader Rust (multi-select)

EDR / Hook Evasion

A

Direct Syscalls (SysWhispers3-style)

Bypass EDR hooks dengan memanggil syscall langsung via SSN — tidak melewati ntdll.dll yang di-hook.

B

NTDLL Unhooking (Disk / KnownDLLs)

Baca versi bersih ntdll.dll dari disk atau KnownDLLs, timpa .text section yang di-hook EDR.

C

Module Stomping

Map shellcode ke ruang memori modul legit yang sudah di-load (bukan MEM_PRIVATE), menghindari deteksi berbasis memory scan.

Memory OPSEC

D

Sleep Encryption (Ekko/Foliage)

Enkripsi seluruh region shellcode di memori saat sleep — mencegah memory scanner menemukan payload saat idle.

E

Return Address / Stack Spoofing

Spoof call stack agar tidak terlihat berasal dari shellcode — bypass stack-based detection (Elastic, CrowdStrike).

F

Heap Encryption saat Sleep

Enkripsi heap region saat loader idle agar IOC berbasis heap tidak terdeteksi.

Anti-Sandbox / Anti-Analysis

G

Domain-Joined Check

Hanya eksekusi jika mesin bergabung ke domain Active Directory — sandbox biasanya tidak di-domain.

H

User Interaction Check

Periksa mouse movement, jumlah proses, uptime, dan RAM sebelum eksekusi.

I

PPID Spoofing

Spawn proses dengan parent process palsu (misal: explorer.exe) agar proses tidak tampak mencurigakan di process tree.

Bypass AV/EDR Instrumentation

J

AMSI Patch

Patch AmsiScanBuffer di memori untuk menonaktifkan AMSI scanning.

K

ETW Patch

Patch EtwEventWrite untuk mencegah event tracing ke EDR/Windows Defender.

L

Hardware Breakpoint AMSI/ETW Bypass

Gunakan hardware debug registers (DR0-DR3) untuk intercept dan bypass AMSI/ETW tanpa modifikasi memori — lebih stealth dari patch.

Payload & PE OPSEC

M

PE Metadata Spoofing + Code Signing

Fake version info, company name, cert cloning dari binary legit (mirip fitur limelighter yang ada).

N

String & Import Obfuscation

Enkripsi string literal dan sembunyikan Windows API import menggunakan dynamic resolution (GetProcAddress hash).

O

Staged Payload (download dari URL)

Payload tidak embed di loader — diunduh saat runtime dari URL yang dikonfigurasi, enkripsi AES/ChaCha20.
