Tipe Output Loader yang Didukung

Pilih semua tipe loader yang ingin dihasilkan oleh web app ini (multi-select)

1

Binary (.exe)

Standalone executable Windows. Shellcode dijalankan langsung dalam proses sendiri.

2

DLL (.dll)

Dynamic library dengan export DllRegisterServer / DllGetClassObject. Dipakai untuk AppDomain injection dan sideloading.

3

AppDomain Loader + .config

DLL khusus yang memuat .NET assembly ke AppDomain proses target. Disertai file .config untuk CLR version dan runtime policy.

4

Shellcode Injector (Process Injection)

Inject shellcode ke proses Windows yang sudah berjalan via NtQueueApcThreadEx / VirtualAllocEx.
